# RecruitVerse Project Report

## Title

RecruitVerse — AI Powered Recruitment Intelligence System

---

## Problem Statement

Recruiters manually screen many resumes, which is time-consuming and error-prone. RecruitVerse automates resume parsing, skill extraction, candidate matching, and ranking.

---

## Objectives

- Extract text from resumes
- Identify candidate skills
- Match resumes with job descriptions
- Rank candidates
- Provide recruiter authentication
- Expose backend APIs
- Support Docker deployment
- Add testing and validation

---

## Modules

1. Resume Parser
2. Skill Extractor
3. Matching Engine
4. Ranking Engine
5. Authentication
6. FastAPI Backend
7. Streamlit UI
8. PostgreSQL Database
9. Docker Deployment
10. Testing and Logging

---

## Technology Stack

- Python
- FastAPI
- Streamlit
- PostgreSQL
- Docker
- PyMuPDF
- bcrypt
- pytest

---

## Testing

Testing was performed using pytest.

Tested modules:

- Parser
- Skill extraction
- Match score
- Ranking
- Authentication
- API health endpoint
- Database connection
- End-to-end resume pipeline

---

## Conclusion

RecruitVerse provides a working recruitment intelligence system with authentication, resume processing, candidate ranking, API backend, Docker deployment, and automated testing.
